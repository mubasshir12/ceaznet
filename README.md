# Kalina AI

Kalina AI is a sleek, modern, and highly interactive web application that serves as a powerful front-end for Google's Gemini API. It offers a rich, conversational AI experience with a focus on user experience, performance, and a wide array of intelligent features.

The application is designed to be a versatile tool for both casual users and developers, providing tools for simple Q&A, complex problem-solving, creative content generation, and real-time voice conversations.

## ✨ Core Features

- **Conversational Chat Interface**: A clean and intuitive chat UI for interacting with the Gemini models.
- **Real-time Streaming**: Responses from the AI are streamed word-by-word, creating a dynamic and engaging user experience.
- **Voice Conversations**: A dedicated, immersive interface for real-time, low-latency voice chats with Gemini, complete with voice selection and persona customization.
- **Optional User Authentication**: Integrated with **Supabase**, allowing users to sign up, log in, and have their profile data persisted across devices. Anonymous usage is fully supported.
- **Long-Term Memory (LTM)**: The AI learns and remembers key facts about the user over time, leading to more personalized and context-aware conversations.
- **Intelligent Tool Selection**: Kalina AI automatically analyzes user prompts to decide the best tool for the job, including:
    - **Web Search**: For real-time information, current events, and up-to-date topics.
    - **URL Reader**: Fetches and analyzes content from web pages.
    - **Chemistry Tool**: Visualizes 3D models of chemical compounds.
    - **Thinking Mode**: Shows the AI's step-by-step reasoning process for complex queries.
- **Rich Media Support**: Users can upload images and files (PDF, TXT) to have conversations about them.
- **Advanced UI Components**:
    - **3D Molecule Viewer**: An interactive viewer for chemical structures fetched from PubChem.
    - **Image Editor**: A simple in-app editor for annotating or drawing on images before sending them to the AI.
    - **Code Preview**: A live-preview modal for HTML/CSS/JS code generated by the AI, complete with a console for debugging.
    - **Reading Mode**: A distraction-free view for long conversations.
- **Developer Console**: An in-app console for monitoring token usage, application logs, conversation summaries, and storage details.
- **Usage & Storage Management**: Detailed dashboards to track token usage per conversation and manage the application's storage on the user's device.
- **Persistent State**: Conversations, memory, and settings are saved locally using IndexedDB, ensuring data is retained between sessions.
- **Theming**: Supports light, dark, and system-preference themes.
- **Responsive & Modern Design**: Built with TailwindCSS for a beautiful and responsive experience across all devices.

## 🛠️ Technology Stack

- **Front-end**: React, TypeScript
- **Styling**: TailwindCSS
- **AI**: Google Gemini API via `@google/genai` SDK
- **Authentication & User Profiles**: Supabase
- **Local Storage**: IndexedDB (for conversations, memory, etc.)
- **3D Rendering**: Three.js (for the Molecule Viewer)
- **UI Components**: `lucide-react` for icons

## 🚀 Getting Started

1.  **API Key**: To use the application, you need a Google Gemini API key. You can obtain one from [Google AI Studio](https://aistudio.google.com/app/apikey).
2.  **Supabase Setup**:
    - Create a new project in [Supabase](https://supabase.com/).
    - In the SQL Editor, run the commands provided in the `_sql_*.md` files to set up the database schema and policies. It is recommended to run them in order: `_sql_user_management.md`, `_sql_user_data.md`, and then `_sql_public_content.md`.
    - Obtain your project's URL and anon key.
3.  **Launch the App**: When you first open the app, it will prompt you for your Gemini API key. Enter it to begin. User authentication via Supabase is optional and can be accessed from the header.
